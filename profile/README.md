# DarkGameHub

DarkGameHub 是一个由 4 人组成的独立游戏工作室。

我们关注叙事、合作、悬疑和黑色幽默，正在尝试把日常工作和公共空间做成既荒诞又有压迫感的游戏体验。

## 当前项目

### 黑色外包 / Black Commission

《黑色外包》是一款 1-4 人合作的外包事务所生存恐怖游戏。

玩家在一家快要倒闭的事务所里接下越来越离谱的外包委托，整理装备，前往失控的学校、商场、公寓、诊所等公共空间完成任务，并在异常和催债般的怪物追上来之前撤离。
